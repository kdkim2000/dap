## 1. 사업개요

### 1) 사업목적

DAP(데이터아키텍처 전문가) 자격증 시험을 효율적으로 준비할 수 있도록, 6과목 이론 학습·필기 문제 풀이·실기 연습을 한 곳에서 제공하는 웹 기반 학습 플랫폼을 구축한다.

### 2) 사업개요

기존 DAsP(준전문가, 4과목 50문항) 준비 사이트를 DAP(전문가, 6과목 75문항 필기 + 실기 1문항 40점) 기준으로 전면 개편한다. 필기는 전사아키텍처 이해·데이터 요건 분석·데이터 표준화·데이터 모델링·데이터베이스 설계와 이용·데이터 품질 관리이해 6과목을 커버하며, 실기(논리 데이터 모델 작성 + 표준화 정의서 작성)를 별도 연습 영역으로 신규 제공한다.

### 3) 추진 배경

- 기존 DAsP 사이트는 4과목(14챕터, 50문항 모의고사) 기준으로 설계되어 DAP 시험과 구조가 상이함
- DAP 시험은 6과목(21챕터), 240분, 실기 40점 비중으로 난이도가 높아 별도 학습 지원이 필요
- 특히 실기(논리 데이터 모델·표준화 정의서)는 기존 사이트에 기능이 전혀 없어 신규 구현이 필수

### 4) 사업 내용

- **필기 이론 학습**: 6과목 21챕터 마크다운 이론 콘텐츠 제공 (기존 4과목 14챕터 → 확장)
- **필기 문제 풀이**: 21챕터 단원별 객관식 문제 풀이 및 해설
- **필기 모의고사**: 75문항(4과목 25문항, 나머지 과목 각 10문항), 240분 타이머
- **실기 연습**: 논리 데이터 모델 작성(유형1/2) + 표준화 정의서 작성(엔터티/데이터 표준) 연습 문제
- **학습 관리**: 진도 추적(localStorage), 오답 노트, 북마크, 대시보드, 게임화(XP/스트릭)
- **코드베이스 업그레이드**: 타입·ID 체계·모의고사 로직을 6과목 기준으로 전면 수정
